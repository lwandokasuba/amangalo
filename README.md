## Getting Started

<p>A leader board and tournament creation web app...</p>

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lwandokasuba/amangalo.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the local environment
    ```sh
    npm start
    ```
